## 1. Descripción del proyecto

Este proyecto corresponde al **proyecto final del Módulo 3: Fundamentos de la programación en Python**.

Desarrollé una aplicación de consola que simula el comportamiento básico de un **ecommerce de joyería**.  
Desde la terminal puedo:

- Ver un catálogo de joyas.
- Buscar productos por nombre o categoría.
- Agregar productos a un carrito de compras.
- Ver el carrito y el total a pagar.
- Vaciar el carrito.
- Salir del programa.

Todo está implementado usando solamente contenidos del módulo:
**tipos de datos, condicionales, ciclos, estructuras de datos y funciones**.

---

## 2. Objetivos del trabajo

Con este proyecto busqué:

- Practicar la **lógica básica en Python** en un caso cercano a la vida real (un ecommerce).
- Aplicar **condicionales** (`if`, `elif`, `else`) dentro de un programa completo.
- Utilizar ciclos `while` y `for` para manejar el menú y recorrer el catálogo y el carrito.
- Trabajar con estructuras de datos compuestas: **listas y diccionarios**.
- Organizar el código en **varios archivos `.py`** y utilizar funciones (`def`) para separar responsabilidades.
- Implementar una función que reciba parámetros y **retorne un valor** (calcular el total del carrito).
